## Color Palette (Extra Life brand)

| Token | Hex | Usage |
|-------|-----|-------|
| `--el-green` | `#4CAF50` | Primary actions, CTA buttons, sidebar active, success |
| `--el-green-dark` | `#388E3C` | Button hover, active states |
| `--el-green-light` | `#E8F5E9` | Success badges, light accents, active backgrounds |
| `--el-navy` | `#1B2A4A` | Sidebar background, dark header/footer |
| `--el-navy-light` | `#243656` | Sidebar hover state |
| `--el-white` | `#FFFFFF` | Cards, inputs, content areas |
| `--el-light` | `#F5F7FA` | Page background |
| `--el-text` | `#2D3748` | Body text |
| `--el-text-muted` | `#718096` | Secondary text, labels |
| `--el-border` | `#E2E8F0` | Card borders, dividers |
| `--el-danger` | `#E53E3E` | Error states, destructive actions |
| `--el-warning` | `#D69E2E` | Warning badges, pending states |
| `--el-info` | `#3182CE` | Info badges, links |
