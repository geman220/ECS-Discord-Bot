Responsive Tables Demo

Demonstration of mobile-friendly responsive table patterns

Overview

This responsive tables solution converts standard data tables to card-based layouts on mobile devices, making it easier for users to view and interact with tabular data on small screens.

Key Features
  • Automatically transforms tables to cards on mobile
  • Displays column headers as labels on mobile
  • Special handling for action buttons
  • Proper styling for dark mode
  • Compatible with existing table styles
  • Works with DataTables plugin
How to Use
  1. Add .mobile-card-table class to your table
  2. Add a table header with column names
  3. JS will automatically add data-label attributes
  4. Use existing Bootstrap table classes as needed
Basic Example

Implementation Guide
How to update your existing tables

To convert any table in your application to use this responsive card pattern:

  1. Add the mobile-card-table class to your table element
  2. Ensure your table has proper <thead> and <tbody> sections
  3. For tables with action buttons, no additional changes needed
  4. For DataTables, you may need to adjust responsive options
Example Code
<!-- Basic Implementation --> <div class="table-responsive"> <table class="table table-striped mobile-card-table"> <thead> <tr> <th>Name</th> <th>Email</th> <th>Phone</th> <th>Actions</th> </tr> </thead> <tbody> <tr> <td>John Doe</td> <td>john@example.com</td> <td>555-1234</td> <td> <a href="#" class="btn btn-sm btn-primary">View</a> <button class="btn btn-sm btn-danger">Delete</button> </td> </tr> </tbody> </table> </div>
