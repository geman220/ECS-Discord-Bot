    Import Schedule from CSV

    Upload a CSV file with match schedule

    Expected CSV Format

    Your CSV should have the following columns (header row required). Multiple date/time formats are supported:

    Column Required Format Example
    Opponent Yes Text or "Bye" Ice Crushers
    Date Yes M/D/YYYY, MM/DD/YYYY, or YYYY-MM-DD 1/7/2026
    Time Required* H:MM AM/PM or HH:MM 8:00 PM
    Location Optional Text Miller Playfield
    Home or away Optional Home/Away or Y/N Away
    Shirt color Optional Text (your team's color) Black
    Opponent shirt color Optional Text (opponent's color) White
    Notes Optional Text 60/40 Koed for this match

    * Time is optional for Bye weeks only

    Bye Weeks: Set opponent to "Bye" to create a bye week entry (time and location not required).
