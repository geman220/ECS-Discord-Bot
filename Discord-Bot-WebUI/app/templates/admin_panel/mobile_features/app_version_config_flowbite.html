    Quick Reference

    Normal update: Set Latest Build Number to the new build. Keep Force Update off. Users see a soft prompt.

    Breaking change: Set Min Build Number to the new build number. Turn on Force Update. Users on older builds are blocked until they update.

    After rollout stabilizes: Turn Force Update back off.
