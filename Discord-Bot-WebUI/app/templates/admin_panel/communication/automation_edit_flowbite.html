    The trigger type is fixed for a rule — it decides which detector runs.

    Every active team in the league must reach this many non-coach players. Coaches are drafted onto teams about a week early, so a low number here risks firing on the coach draft instead of the real one.

    -3 = three days before, +1 = day after. Counted from local midnight.

    Email on its own uses the email-blast system: your HTML layout, and a per-person record of who received it. Adding any other channel switches to the notification system — genuinely multi-channel, but you only get per-channel totals, not per-person delivery rows.

    SMS is deliberately not offered here: sends are capped at 100/hour across the whole system, so an unattended blast would quietly start failing halfway through.

    Plain text, no HTML — a push notification cannot render it. Leave blank and the HTML body gets stripped to text automatically, which usually reads badly.
