    About Message Templates

    Message Templates allow you to create reusable, customizable messages for the system. Organize templates into categories (e.g., "Notifications", "Welcome Messages", "Match Reminders") and use variables to personalize messages dynamically.

    Example Categories:
    • Match Notifications
    • Welcome/Onboarding
    • Discord Bot Messages
    Example Variables:
    • {player_name}
    • {team_name}
    • {match_date}
    Where Used:
    • Email notifications
    • SMS messages
    • Discord bot announcements
