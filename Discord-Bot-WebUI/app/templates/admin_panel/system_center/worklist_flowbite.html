    How the System Center works

    One operations surface. Overview and Services are live now; Performance, Jobs, Data, Security, Logs and API traffic light up in later phases. Empty tabs mean "no live source wired yet" — never a hidden failure.
    Honesty first. Every figure traces to a real collector. Values that describe only this worker or are estimated carry a small badge, so you always know how much to trust a number.
    Never faked healthy. A service that can't be reached reads Down or Idle with the real reason. Twilio stays Idle until you explicitly test it — we don't probe it on every load.
    Actions are live. Quick actions and per-service controls run for real against the existing operations endpoints, with CSRF protection. Destructive ones (restart bot, flush Redis) sit behind a confirmation and are shown only to Global Admins; every action surfaces the server's own result.
