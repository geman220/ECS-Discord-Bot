How this draft works
Snake order
The pick order reverses every round (E→L, then L→E, then E→L…). Whoever picks last in round 1 picks first in round 2, so no team is stuck picking last every round. This is the fair default.
Linear order
The same order every round (E→L, E→L…). Simpler, but the first team always picks first — a slight advantage each round.
Rounds
