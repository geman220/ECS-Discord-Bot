Responsive Tables Demo
Mobile-friendly table patterns

View this page on a mobile device or narrow your browser window to see how tables transform into card views.

How to use responsive tables:
  1. Add the mobile-card-table class to your table element
  2. Ensure you have proper table headers and data cells
  3. The system will automatically convert to cards on mobile

Implementation Guide
HTML Structure:

<div class="table-responsive">
    <table class="table table-striped mobile-card-table">
        <thead>
            <tr>
                <th>Name</th>
                <th>Email</th>
                <th>Actions</th>
            </tr>
        </thead>
        <tbody>
            <tr>
                <td data-label="Name">John Doe</td>
                <td data-label="Email">john@example.com</td>
                <td data-label="Actions" class="table-actions">
                    <a href="#" class="btn btn-sm btn-primary">View</a>
                    <button class="btn btn-sm btn-danger">Delete</button>
                </td>
            </tr>
        </tbody>
    </table>
</div>
Benefits of this approach:
  • No horizontal scrolling on mobile devices
  • Clear labels for each data field
  • Maintains all your existing table styling
  • Automatically adapts to different screen sizes
  • Works with existing Bootstrap table classes
