Design Tips
Best Practices
  • Use transparency where appropriate in your PNG files
  • Keep text minimal on images - use the pass fields for text
  • Use consistent branding across all your assets
  • Test on different devices to ensure visibility
Image Specifications
  • Format: PNG with transparency where needed
  • Resolution: Use the exact pixel dimensions specified
  • 2x versions: Optional but recommended for high-resolution displays
  • File size: Keep under 1MB for optimal performance
